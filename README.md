# Studio App - Sistema de Agendamento

MVP de sistema de agendamento para studio de pilates/fisioterapia.

## Estrutura do Projeto

```
studioApp.v1/
├── frontend/           # Frontend Vue.js 2 + TypeScript + Vuetify
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── ...
└── backend/            # Backend (futuro)
    └── ...
```

## Frontend

Desenvolvido com Vue.js 2 + TypeScript + Vuetify.

### Instalação e Execução

```bash
cd frontend
npm install
npm run serve
```

### Funcionalidades
- Login com CPF e senha (mockado)
- Dashboard diferenciado (Admin/Cliente)
- Sistema de agendamentos
- Interface acessível para pessoas mais velhas
